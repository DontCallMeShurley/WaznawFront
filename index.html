Описание проекта

Мощный сервис для массовых рассылок покупателям маркетплейсов. Поддерживает сегментацию аудитории, персонализацию сообщений и гибкую систему триггеров. Интеграция с OZON позволяет автоматизировать маркетинговые кампании типа "купил товар А - получи промокод на товар Б".

Ключевые функции

• Массовые рассылки с персонализацией
• Интеграция с маркетплейсами
• Триггерные кампании и автоворонки
• Детальная аналитика конверсий

Технологический стек

.NET Core PostgreSQL DevExpress RabbitMQ Docker Redis

Online
Дата запуска 10 января 2023

Описание проекта

Комплексная платформа облачных решений для бизнеса. Включает виртуальные серверы с автомасштабированием, управляемые базы данных, объектное хранилище S3-совместимое и глобальную CDN. Обеспечивает 99.9% доступность сервисов.

Преимущества платформы

• Автоматическое масштабирование под нагрузку
• Резервное копирование и восстановление
• DDoS-защита и Web Application Firewall
• Мониторинг и алерты в реальном времени
• Pay-as-you-go модель оплаты

Weather App

Online
Дата запуска 1 октября 2024
Демо версия weather.waznaw.ru

Описание проекта

Современное погодное приложение с интуитивным интерфейсом для отслеживания погоды в любой точке мира. Использует открытые данные Open-Meteo API, что позволяет работать без ограничений и API-ключей. Полностью адаптивный дизайн обеспечивает комфортное использование на любых устройствах.

Функциональные возможности

• Текущая погода с детальной информацией
• Почасовой прогноз на 24 часа с графиками
• Недельный прогноз с трендами
• История поисковых запросов за неделю
• Автоопределение местоположения
• Темная и светлая темы оформления
• Автодополнение при поиске городов
• Офлайн-режим с кешированием данных

Технологический стек

HTML5 CSS3 JavaScript ES6+ Open-Meteo API LocalStorage Docker PWA
